# 🏍 kawasaki Bike Chatbot

A Streamlit-based chatbot that allows users to search, filter, and sort Kawasaki motorcycle models (2000–2026 range dataset).

## 🚀 Features

- Search Kawasaki bikes by name, cc, or specs
- Filter by HP, torque, weight, speed, fuel efficiency
- Sort results (high → low or low → high)
- Clean UI with Kawasaki branding and background wallpaper
- Dataset-based chatbot (fast and offline)

---

## 📊 Dataset

Includes Kawasaki bike specifications:

- Model
- Year
- Engine CC
- Horsepower (HP)
- Torque (Nm)
- Weight (kg)
- Top Speed (km/h)
- Fuel Average (km/l)
- Gears
- Fuel Type
- Price (optional)
- Color /
